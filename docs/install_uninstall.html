Install YourNotes

YourNotes is an open-source application first released on Github. Several ways are provided to install YourNotes for different platforms as follows.


For Windows

1.Installation package(Recommended)

Download the installation package online, and double-click to run the setup program and follow the instructions to finish the software installation. You'll need to select the target directory and program group name during installation.


2.Source code

Visit YourNotes repository page and fork YourNotes to your own repositories. Then pull it to the local disk and compile the source code with Qt Creator.

We don't recommend this way unless you are familiar with git, Github as well as Qt 5.x and Qt Creator.

If you decide to compile the source code by yourself, please note that YourNotes was developed with Qt 5.5.1 without consideration of the compatibility with Qt 4.


For Mac OS X

1.Installation package(Recommended)

Open the downloaded .dmg package file, then drag the YourNotes icon and drop it into the system's Applications folder, whereby you can double click on the program's icon to start it.


2.Source code

Pull the source code to the local and compile it by yourself. Refer to the steps for Windows for more details.


For Linux

1.Source code

We only provide installation by source code for Linux currently. Pull the source code to the local and compile it by yourself. Refer to the steps for Windows for more details.


Uninstall YourNotes

We have different ways to removed YourNotes from your PC, depending on your operating system and installation method.


For Windows

If you installed YourNotes by intallation package, you may need to select the [Uninstall YourNotes Desktop] menu item from the Start menu, and press the "Yes" button to proceed.

If you choosed to use YourNotes by source code, you may need to find the directories containing the source code and executable file, and trash them.


For Mac OS X

If you installed YourNotes by intallation package, simply trash the program icon from the system's Applications folder.

If you choosed to use YourNotes by source code, you may need to find the directories containing the source code and executable file, and trash them.


For Linux

Find the directories containing the source code and executable file, and removed them.
