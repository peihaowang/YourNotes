Formatting and Editing Text

YourNotes provides full-featured text editing and formatting.


Editing


1.Undo/Redo

Click [Undo/Redo] in the right-clicked menu and shortcut keys are also supported.


2.Cut/Copy/Paste/Select

Click [Cut/Copy/Paste/Select All] in the right-clicked menu and shortcut keys are also supported.



Formatting


1.Bold/Italic/Underline/StrikeOut

Click [Bold/Italic/Underline/StrikeOut] menu item in the right-clicked menu to apply the format to the current selected text. Shortcut keys are also supported.


2.Font

Click [Font...] menu item in the right-clicked menu and fill the popup dialog to custom the font of the current selected text.


3.Highlight Color

Select [Highlight...] in the right-clicked menu and pick out a color set as the foreground color of the current selected text.



Customize the Initial Format

Right-click YourNotes tray icon on the system task bar and select [Options...] menu item, then customize the options under the "Notes" category.
