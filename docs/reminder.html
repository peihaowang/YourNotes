What is Reminder?

YourNotes provides reminder features for schedule reminding. You can set an alarm time for a note page, and then YourNotes will remind you when the note is due.



Set Reminder

Right-click at a note page and select [Set Reminder...] menu item and set the alarm time in the dialog.

Please note that the alarm time should not be ealier than present.



Cancel Reminder

Simply right-click at a note page and select [Cancel Reminder...] menu item.



How Does Reminder Work?

When a note is due, the note page will start to make some reactions to alarm you, such as shocking the page itself strongly and playing a sound.

Once the note page obtains the focus again, it will stop alarming.



Reminder Options

Right-click YourNotes tray icon on the system task bar and select [Options...] menu item, then customize the options under the "Reminder" category.


1.Sound

Choose preferred audio for your alarming sound.


2.Interval

If you choose [Forever], the note page will shock itself and play sound ceaselessly until you focus on the note page.

Otherwise, the note page will alarm for about 15 seconds and have a interval before next alarm.


3.Shock Factor

This option determines how strong the note page shocks itself to remind you.
