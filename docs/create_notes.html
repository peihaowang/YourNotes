Create Notes

When you open YourNotes at the first time, nothing will be displayed on the screen, but a tray on the task bar, which means YourNotes launches successfully. You need to create your first note simply by right-click the tray icon and select [New Note] menu item, then a note page will be displayed on the top-right corner of your screen.

Since you already have a note pinned on the screen, you can create a new note simply by clicking the [Add] button on the top-left corner of the note page.

Also, you can click [New Note] in the right-click menu to create new note.



Trash Notes

When you don't need a note anymore, you can drop it by clicking the [Cross] button on the top-right corner of the note page.

Note that this operation doesn't mean you removed the note permanently. YourNotes just hide the note and you can find and restore in the [Manage Notes] window.

Also, you can click [Delete Note] in the right-click menu to trash the current note.

Please note that YourNotes will not keep an an empty notes after you trash them.
