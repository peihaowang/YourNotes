Manage Notes

YourNotes provides the note management window for batch operations.


Right-click YourNotes tray icon on the system task bar, and select [Manage Notes...] menu item to open note management window.

You can browse the overview and created date of each note and check or uncheck to modify the attributes of notes.


1.Untrash Notes

Check the box in the "Display" column to display the hidden notes. Also, uncheck the box can hide the visible notes.



2.Read Only Attribute

Check/uncheck the box in the "Read Only" column to set the read only attribute.
