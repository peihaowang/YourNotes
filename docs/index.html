Introducation to YourNotes

YourNotes is a simple and quick mini-program for personal note managment and schedule reminding. All notes created by YourNotes is stored on the disk and can be displayed or hidden on the screen. YourNotes is built for efficiency, practicality and convenience as an open-source application. And this version is specifically developed for desktop usage with Qt framework. Now YourNotes Desktop supports cross-platform feature and is available for Windows XP/Vista/7/8/10, Mac OS X and Linux.


Documentation of YourNotes

YourNotes documentation is written by the author of YourNotes and provides information about operation instructions and matters needing attention. The documentation consists of two parts: the profile of YourNotes(shown on this page) and detail instructions.

Since YourNotes documentation only provides limited information, you may consult us via email wangpeihao@gmail.com if you meet any problem which cannot be solved by yourself.



About YourNotes

The predecessor ofYourNotes, Note Box, was developed in 2015, but not released to the public. We only submitted the source code to a private repository on Bitbucket. We used a third-party storage engine is that version, however, the storage engine doesn't support any open-source licenses. So we rewrote the storage part in January, 2018. Although the new version discarded some features on personal information protection and data size optimization, it became an open-source project finally. We also fixed some bugs reserved from the old versions and enhanced some features. Then we used "YourNotes" as the new name of the newly released open-source version.



From the Author

The author of YourNotes(also the author of the documentation) is Peihao Wang, an undergraduate from ShanghaiTech University. The development of the first generation of YourNotes(Note Box) is finished when I was still in senior high school, and I stopped maintaining the project because of the participation of the development of another commercial project and the preparation of the college entrance examination. And in the winter of 2018, I decided to release an open-source version of this useful tool. So I refactored the source code and rewrote the storage modules, making the new version completely available for open-source release.

The inspiration of YourNotes was from my daily life. There was always a need to pin something on the screen when I was using PC, so I first developed the tool for my own usage. Then I enhanced some features and made it a product. The first name of YourNotes was Note Box, and I renamed it as YourNotes for the open-source release.
